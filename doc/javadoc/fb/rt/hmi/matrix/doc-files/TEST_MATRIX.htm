SYSTEM TEST_MATRIX

This system configuration provides facilities for testing operations on instances of the MATRIX data type.

The initial situation after launching the test configuration.
The situation after entering data in the [0,0] position of the IN1 input.
Note the error codes of 4 (INVALID_ARRAY_SIZE) and 13 (DIVISION_BY_ZERO).
The situation after entering data in the [0,1] position of the IN2 input. Note the error codes of 4 (INVALID_ARRAY_SIZE) have disappeared since consistently-sized arrays are now available for the ADD, SUB and MUL functions. However, the 13 (DIVISION_BY_ZERO) error code is still present because IN1 is singular.
The situation after entering data in the [2,2] position of the IN1 input. Now the 13 (DIVISION_BY_ZERO) error code is gone because IN1 is non-singular.
The situation just before entering invalid data in the [0,0] position of the IN1 input. Upon attampting to enter the invalid data, the value reverts to its previous value of 1.0, i.e., invalid data is not accepted.
