Identification

This dialog lets you enter information that may be used for retrieval of the library element from a data base. You can open an item for editing by double-clicking on its position in the second column, and enter its value by pressing the Enter key or by clicking on another item to enter. You can save the edited values by clicking on the OK button, or discard them and close the window by clicking on the Cancel button.

  • Standard: The primary reference standard for this library element in number-part-subclause format.
  • Classification: Classification code*
  • ApplicationDomain: Application domain*
  • Function: Function of this element*
  • Type: The element type (e.g., device type)*
  • Description: Descriptive phrase*
* As defined in the reference standard, if applicable.
