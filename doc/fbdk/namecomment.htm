Name/Comment

This dialog permits editing of the name and comment information for the library element. You can open a field for editing by double-clicking on it.
 You can save the edited results by pressing the Enter key while editing a field, or by pressing the OK button. An error message is given in the status line if any of the entered information is invalid .
