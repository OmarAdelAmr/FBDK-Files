What's New

This page documents changes since the 20050311 release of the FBDK.

FBDK [FBRT] [MVCD]

  • This is the first maintenance release of the FBDK by Holobloc, Inc. under license from Rockwell Automation.
  • The FBDK has been tested to work with the J2SE 1.4.2 SDK and the J2SE 5.0 SDK, which has a more "modern" look and feel.
  • Initialization of remote devices now uses the host IP address instead of localhost for the management port.
  • Source files in the src subdirectory hierarchy have been scanned with the XMLChecker application and updated for compliance with the current XML DTDs defined by IEC 61499-2.
  • Implementation of the File Save As HTML option has been updated so that emitted code works better with current JVM implementations.
  • Implementation of the File Save As XML option has been updated to emit correct syntax for resource types and device types.

FBRT [FBDK] [MVCD]

  • The FBRT has been tested to work with the J2SE 5.0 SDK and may not work with earlier versions of the SDK.
  • Various details of the FBRT implementation have been modified for future compatibility with the J2ME CLDC profile for embedded devices.
  • Errors in the initialization of variables of type LINT and TIME from strings have been corrected.
  • Many small improvements have been made in efficiency of the runtime classes.

MVCD [FBDK] [FBRT]

  • The DRIVE function block type is no longer used. Its timing function has been replaced by a clock pulse distribution mechanism using PUBL_0/SUBL_0 blocks, and its motion simulation function has been integrated into models, e.g., MECH2_MDL.
Last updated:  2005-05-21.
