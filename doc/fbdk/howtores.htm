Build and Test a Resource Type

  1. Start the FBEditor application by running the editor.bat file.
  2. Create a new resource type in the editor by:
    • Opening a file containing an existing resource type, such as the RMT_RES.res file in the src\ita folder or the TEST_RES2.res file in the src/test folder; or
    • Clicking the File => New => ResourceType menu item. Here is how the screen will look if you do this:
  3. Edit the resource type's elements:
    • Edit the resource type's properties.
    • Edit the resource type's interface (data inputs, i.e., parameters, only).
    • Edit the resource type's body by clicking on the resource type name in the Navigation area and editing the graphic layout of the FBD in the Worksheet/Viewport area.
    • Edit the FB types supported by the resource.
  4. When your editing is complete, save your work with the File=>Save As=>XML... menu item.
    • It is a good idea to save your work from time to time while editing, too!
  5. Generate and compile a JavaTM source file with the File=>Save As=>Java... menu item.
  6. Resources cannot be tested independently of their containing devices, so you will have to create a system configuration to do this testing. For an example see the TEST_SYS2 configuration.
