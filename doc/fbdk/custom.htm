The procedures described below have only been tested with the Windows™ XP operating system and may have to be modified for other operating systems.

Starting the FBEditor

  • After the JDK™ 6 has been installed, the FBEditor can be started by double-clicking on the icon of the editor.jar file or selecting the Open option of its pop-up menu.
  • The FBEditor can also be started by double-clicking on the icon of the fbeditor.bat file or selecting the Open option of its pop-up menu.
  • The fbeditor.bat file can also be invoked in a command prompt window.
  • Files containing IEC 61499 library elements, e.g., *.fbt and *.res files, can be opened by double-clicking by associating their file types with the fbeditor.bat file.
  • IEC 61499 library files can also be opened by dragging their icons onto the icon of the fbeditor.bat file.
  • If fbeditor.bat is not located in an fbdk subdirectory of your home directory, e.g., C:\fbdk, you should edit the first line of fbeditor.bat to change the working directory to the home of the fbeditor.bat file.

The editor.ini File

Each line of this file consists of a parameter name followed by a parameter value. The allowed parameter names and values are given in the following table. If no value for a parameter is given in the editor.ini file the default value applies. Different values for these parameters will need to be given for different Java™ Development Kit (JDK) and Java Runtime Environment (JRE) installations and for non-Windows operating systems.

Parameter Name Explanation Default
browser The command to run the Web browser for help files. \\Program Files\\Mozilla Firefox\\firefox
java The command to run the Java™ Runtime Environment (JRE) executable. java
javac The command to execute the Java compiler. \\Program Files\\Java\\jdk1.6.0_03\\bin\\javac
timeout The initial timeout period for management commands to remote devices. t#2s
