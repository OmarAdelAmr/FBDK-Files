What's New

This page documents changes since the 20060110 release of the FBDK.

FBDK [FBRT] [MVCD]

  • FBEditor is now a single window with multiple tabs instead of multiple windows, simplifying navigation when multiple library elements are simultaneously open for editing.
  • The FBEditor application is now directly launchable by double-clicking on the editor.jar file.
  • Multiple configuration files are now consolidated into a single editor.ini file.
  • Configuration bug fixes since the 20060202 version:
    • Documentation errors about how to set the javac location have been fixed.
    • An error in initializing the path configuration parameter is fixed.
    • The doc and javadest configuration parameters are deleted; these are at the immutable locations ./doc and ./lib/fb/rt relative to the FBDK home directory, respectively.
    • Help functions should now work with both Firefox® and Internet Explorer™ (when properly configured).

FBRT [FBDK] [MVCD]

  • No significant changes since 20050521.

MVCD [FBDK] [FBRT]

  • No significant changes since 20050521.
